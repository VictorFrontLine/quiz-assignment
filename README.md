# 실행 환경

-   node >= 16

# 설치한 패키지

## React

-   Create React App

## Linter/Formatter

-   eslint
-   prettier

# 설정 파일

-   `.eslintrc.json`
    - plugin 추가 (react, prettier)
    - prettier/prettier를 오류로 인식
-   `jsconfig.json` : 절대경로로 입력하면 `src` 기준으로 resolve 하도록 설정
-   `.prettierrc` : Prettier 세부 설정
